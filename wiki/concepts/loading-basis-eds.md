# Loading Basis through EDS

Procedure for loading cost basis data for plan participants through EDS.

## Current Documentation

The EDS hub page covers general EDS layout setup, validation, and processing applicable to all data types including basis:

> **[[eds]] — see Key Functions, Layout Setup, and Output sections**

Basis loading uses the same EDS framework as other data types:
- DC sets up the basis layout in EDS (P3 menu bar → EDS) before files arrive
- Files can be CSV, Excel, or fixed-width
- EDS output (errors and warnings) must be reviewed after every run
- Multiple layouts can be active simultaneously — basis can be set up alongside other layouts
