# VEF — Vendor Expense Form

A document used when a vendor expense arrangement needs to be established for a plan. Routed by the [[com]] through an advisor firm, the plan sponsor, and then submitted for internal L&C (Licensing and Compliance?) setup via Salesforce.

## Definition

Some plans involve expense or fee arrangements with vendors that require formal documentation and system setup. The VEF is the form that captures and authorizes these arrangements. It is marked "if applicable" in the COM checklist — not every conversion requires one.

## Workflow

1. COM sends VEF to advisor firm for completion and approval
2. Advisor approves the form
3. COM sends approved form to client for signature
4. COM receives signed copy
5. COM submits a ticket to L&C via SF (Salesforce?) to set up the arrangement in the system
6. COM receives confirmation that setup is complete

## Open Questions

- What "L&C" stands for — likely Licensing and Compliance, but not confirmed
