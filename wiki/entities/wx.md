# WX — Plan Sponsor Site

Transamerica's web-based portal for plan sponsors. WX (also called "WX V2" when referencing payroll integration) is set up by [[tc]] and verified by [[com]].

## Overview

WX is the plan sponsor-facing web platform that allows plan administrators to manage their retirement plan online. It includes payroll file upload functionality in its V2 iteration. Setup is owned by TC; the COM verifies that it is configured correctly and follows up if it is not.

## Key Facts

- "WX" likely stands for a Workplace Experience portal or similar — full name not confirmed
- "WX V2" refers to a version with payroll functionality; COM must confirm early whether a plan needs this
- TC is responsible for setting up WX access for the plan sponsor
- COM confirms setup and requests it from TC if not yet done

## Role in Plan Conversions

1. COM confirms early whether the plan needs payroll functionality (WX V2)
2. COM checks whether the plan is set up on WX
3. If not set up, COM emails TC to request WX access setup
4. TC sets up WX; plan sponsor can then access the portal
