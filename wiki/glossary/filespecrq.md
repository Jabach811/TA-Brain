# FILESPECRQ

**Full Name:** File Specification Request

## Definition

FILESPECRQ is an [[awd]] ticket type used by the [[lm-dc]] to hand off incomplete payroll testing to Fiduciary Services at go-live. When payroll testing is not finished by the plan's go-live date, the DC submits a FILESPECRQ ticket containing the payroll vendor contact, client contact, file layout, and a summary of the current testing status. Fiduciary Services then takes over the payroll testing project using their Inforce Payroll Projects process.

This procedure became effective February 26, 2026.

## Used By

- [[lm-dc]] — submits the ticket at go-live when payroll is not yet complete
