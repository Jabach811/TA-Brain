# Document Operations (Doc Ops)

Sets up P3 with new plan details and manages plan documents; owns special plan class additions.

## What This Role Does

Doc Ops updates P3 with new plan information and is responsible for maintaining plan documents. Critically, they are the only team that can add new special plan classes in P3 — DCs can update existing class names and codes but cannot create new special ones.
